This section shows us some examples of default display types.

I am a div. I have a default display type of "block". I also have a maximum width and have been centred using margin: auto!

I am a paragraph. I have a default display type of block. I am a span inside a paragraph. See how I sit inline? There are a few other elements like this - including the strong element. And also LINKS!

I am another paragraph. Because of my default display type, I start on a brand new line.

Next we have some cat images. They are sitting side by side, but have a default display type of inline-block instead of inline, so we can add some top and bottom padding and margins.

Unordered and Ordered lists have default display types of block.
